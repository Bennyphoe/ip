# User Guide

## Features 

### Feature 1
Add Tasks:
* Todo
* Deadline
* Event

### Feature 2
List Tasks

### Feature 3
Set task as done

### Feature 4
Delete Task

### Feature 5
Find tasks with specific keyword

### Feature 6
Sort Tasks chronologically (Deadline only)

### Feature 7
Save Tasks


## Usage

### `Keyword` - Describe action

Describe action and its outcome.

Example of usage: 

1) todo return book

2) list

Expected outcome:

1) Got it. I've added this task:  
[T][ ] return book  
Now you have 1 tasks in the list  

   
2) Here are the tasks in your list:  
 1.[T][ ] return book
